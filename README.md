# Транскрибатор аудио/видео

Локальная транскрибация аудио- и видеофайлов в документ Word (.docx) с таймкодами.

<img width="865" height="694" alt="image" src="https://github.com/user-attachments/assets/377c9f57-90f2-4475-85bd-3247e0cba65f" />


## Возможности

- Поддержка любых аудио/видео форматов (MP3, WAV, FLAC, MP4, AVI, MKV и др.) через ffmpeg
- Транскрибация на русском языке (локально, без API)
- Выбор модели: tiny / base / small / medium / large
- Экспорт в .docx с таймкодами [ЧЧ:ММ:СС]
- Графический интерфейс (Tkinter)

## Установка

**Требования:** Python 3.8+, ffmpeg

```powershell
pip install -r requirements.txt
```

## Запуск

```powershell
python transcriber_gui.py
```

## Модели

| Модель | Точность | Скорость | Память |
|--------|----------|----------|--------|
| tiny   | низкая   | быстрая  | ~1 ГБ |
| base   | средняя  | средняя  | ~1 ГБ |
| small  | хорошая  | медленная| ~2 ГБ |
| medium | высокая  | медленная| ~5 ГБ |
| large  | лучшая   | медленная| ~10 ГБ |

При первом запуске выбранная модель скачивается автоматически.
